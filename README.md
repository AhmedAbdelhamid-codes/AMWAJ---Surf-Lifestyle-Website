# 🌊 AMWAJ - Surf & Beach Lifestyle Website

AMWAJ is a modern and fully responsive front-end website inspired by the spirit of the ocean and surf culture.
The project focuses on clean UI design, smooth user experience, and responsive layouts across all devices.

---

## 📌 Features

* 🌐 Fully Responsive Design (Mobile, Tablet, Desktop)
* 🎨 Modern UI inspired by ocean and surf lifestyle
* 🧩 Structured Semantic HTML
* 💡 Clean and reusable CSS architecture
* 🖱️ Interactive hover effects (cards, buttons, navbar)
* 🔄 Smooth scrolling navigation
* 🎴 Flip card animation (About section)

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox, Media Queries, Transitions, Transformations)
* Font Awesome (Icons)
* Google Fonts

---

## 📂 Sections Included

* 🏠 Home (Hero Section)
* 🛍️ Products (Apparel & Surfboards)
* 🌊 Services (Travel destinations)
* 📖 About Us
* 📞 Contact Form
* 🔗 Footer (Social Media Links)

---

## 🎯 What I Learned

* Building responsive layouts using Flexbox
* Improving UI/UX design thinking
* Handling spacing without breaking layout
* Using CSS animations and transitions effectively
* Structuring a full multi-section landing page

---

## 📸 Design Inspiration

This project was partially inspired by a design found on Pinterest for the hero section, while the rest of the sections were fully designed and implemented by me.
